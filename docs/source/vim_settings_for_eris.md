# Vim settings for development on ERIS

If you're using `vim` please add the following lines to your `~/.vimrc`.

```sh
vi ~/.vimrc
```

```vimrc
filetype plugin indent on
syntax on set number
set incsearch
set ignorecase
set smartcase
set hlsearch
set tabstop=4
set softtabstop=0
set shiftwidth=4
set expandtab
set nobackup
set noswapfile
set nowrap
set colorcolumn=79
```
